# 🚗 Car Price Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on a car price dataset to identify patterns, trends, and factors affecting car prices.

---

## 📂 Dataset
The dataset contains information about:
- Car Name
- Selling Price
- Fuel Type
- Transmission
- Mileage
- Engine Capacity
- Year of Manufacture

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 EDA Techniques Used
- Data Cleaning
- Handling Missing Values
- Duplicate Removal
- Statistical Summary
- Correlation Analysis
- Data Visualization

---

## 📈 Visualizations
- Histogram
- Boxplot
- Scatter Plot
- Correlation Heatmap
- Countplot

---

## 🔍 Key Insights
- Engine size strongly affects selling price
- Newer cars tend to have higher prices
- Petrol cars dominate the dataset
- Mileage impacts resale value

---

## 🎯 Conclusion
The project demonstrates how EDA helps uncover meaningful insights and relationships within real-world datasets.
